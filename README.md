## Run Locally

Clone the project

```bash
git clone https://github.com/yourusername/register-karo-homepage.git
cd register-karo-homepage
```

Go to the project directory

```bash
cd register-karo-homepage
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run start
```
